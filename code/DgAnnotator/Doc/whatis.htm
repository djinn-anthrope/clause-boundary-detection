What is DGA

The Dependency Grammar Annotator (DGA) is a tool for annotating texts with syntactic information according to the formal framework of Dependency Grammars, in particular "incorporating into the text indicators of the syntactic structure: eg. labeled bracketing, or symbols indicating dependency relations between words" (EAGLES) .

Annotated text corpora are quite useful in linguistic practice (testing grammatical theories, automatic acquisition of grammars etc.), but are costly to produce since syntactic annotation is a labor intensive activity. DGA helps in reducing the human effort required in manually annotating documents for the creation of corpora.

DGA provides a simple graphical user interface which allows creating and manipulating easily the syntactic structures of sentences. DGA is based on the formalism of Dependency Grammars, hence the syntactic structures consist in dependency relations between the words of a sentence. Dependency relations are represented graphically as arcs from the dependent word to its head word, labeled with the name of the relation existing between the words. Such a graphical representation conforms to the EAGLES recommendations and is used by DGA as the basis of the annotation operation.

During the annotation process the user acts directly upon this graphical representation. Besides the convenience of usage, the accuracy of annotation increases, since the user receives an immediate graphical feedback regarding any changes he performs on the syntactic structure. Operating upon the syntactic structure is extremely easy and intuitive: in order to create a dependency relation only two mouse clicks are necessary (on the two words involved in the relation which is being created). For assignig a label to an arc, the user needs just to select the label from the list in the left panel. For assignig a part of speech (POS) to a word, one can right click on the word to obtain a popup menu from which to choose the appropriate POS. DGA therefore allows a quick text annotation.

DGA fulfills the requirements stated by [Marcus et al., 1993] for the annotation process:

  • Accuracy - since one operates directly with the graphical representation and because DGA provides immediate graphical feedback to the user.

  • Speed - creation and handling of the dependency relations are quick mouse operations.

  • Consistency - the sets of parts of speech and dependency relations are established by the user after which, in order for them to be used, it is only necessary to select them from various lists.
